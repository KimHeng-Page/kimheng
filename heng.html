Web developement


A web developer is a professional who designs, builds, and maintains websites and web applications. They translate user stories, mockups, and web designs into functional websites using web fundamentals like HTML, CSS, and JavaScript. Web developers also work with graphic designers to create the user interface and may collaborate with other professionals to create and test applications.

UX/UI Design


A UI/UX designer focuses on creating digital products, like websites and apps, that are both visually appealing and user-friendly. They combine elements of visual design, interaction design, and user research to ensure a positive user experience user needs, defining user flows, creating wireframes and prototypes, designing intuitive interfaces, conducting usability testing, and iterating on designs based on user feedback.

Video Editor


A video editor is a professional who uses software to organize video and audio files into a cohesive final product. They are responsible for assembling raw footage, adding effects, and ensuring the video meets the desired vision. Key responsibilities include reviewing footage, making edits, adding audio and visual elements, and collaborating with the production team.
